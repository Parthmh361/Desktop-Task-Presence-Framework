# @dtpf/shared-types

Shared TypeScript types for the [Desktop Task Presence Framework (DTPF)](https://github.com/Parthmh361/Desktop-Task-Presence-Framework).

## Install

```bash
npm install @dtpf/shared-types
```

Usually consumed as a dependency of `@dtpf/sdk-core` or `@dtpf/sdk-react`.
